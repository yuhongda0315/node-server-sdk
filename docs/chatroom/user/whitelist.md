## 聊天室消息白名单模块

### Whitelist.add(chatroom){#add}

添加聊天室用户白名单

`chatroom` 参数说明：

| 参数   	 |	类型		| 必填	| 说明 							|最低版本		|
| :----------|:--------	|:-----	|:------------------------------|:-------- |
|	id	 |	string	|	是	| 聊天室 Id					||
|	userId |	string	|	是 	| 聊天室中用户 Id，可提交多个，聊天室中白名单用户最多不超过 5 个| &nbsp;|

##### 请求成功

```json
{
    "code": 200
}
```

### Whitelist.remove(chatroom){#remove}

删除聊天室用户白名单

`chatroom` 参数说明：

| 参数   	 |	类型		| 必填	| 说明 							|最低版本		|
| :----------|:--------	|:-----	|:------------------------------|:-------- |
|	id	 |	string	|	是	|	聊天室 Id					||
|	userId |	string	|	是 	| 聊天室中用户 Id，可提交多个，聊天室中白名单用户最多不超过 5 个。| &nbsp;|

##### 请求成功

```json
{
    "code": 200
}
```
### Whitelist.getList(){#getList}

获取聊天室用户白名单

##### 请求成功

```json
{
	"code": 200,
	"members":["user1","user2"]
}
```
| 参数   	 |	类型		| 说明 							|最低版本		|
| :----------|:--------	|:------------------------------|:-------- |
|	members	 |	string	| 被禁言用户数组					||
|	time	 |	string	| 解禁时间						||
|	id |	string	| 被禁言用户 Id。				| &nbsp;|
