## 聊天室全局禁言模块

### Ban.add(chatroom){#add}

添加聊天室全局禁言

`chatroom` 参数说明：

| 参数   	 |	类型		| 必填	| 说明 							|最低版本		|
| :----------|:--------	|:-----	|:------------------------------|:-------- |
|	memberIds |	array	|	是 	| 成员列表，最多不超过 20 个		|2.0.0|
|	minute |	number	|	是 	| 禁言时长，范围 1 - 43200 分钟	| 2.0.0|

##### 请求成功

```json
{
    "code": 200
}
```

### Ban.remove(chatroom){#remove}

删除聊天室全局禁言

`chatroom` 参数说明：

| 参数   	 |	类型		| 必填	| 说明 							|最低版本		|
| :----------|:--------	|:-----	|:------------------------------|:-------- |
|	memberIds|	array	|	是 	| 移除多个用户 					| 2.0.0 |

##### 请求成功

```json
{
    "code": 200
}
```
### Ban.getList(){#getList}

获取聊天室全局禁言列表

##### 请求成功

```json
{
	"code": 200,
	"members": [{
		"time": "2015-09-25 16:12:38",
		"id": "2582"
	}]
}
```
| 参数   	 |	类型		| 说明 							
| :----------|:--------	|:------------------------------
|	id 		 |	string	| 被禁言用户 Id。						
|	time	 |	string	| 解禁时间
